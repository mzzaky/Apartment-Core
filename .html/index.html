Get started in
minutes

ApartmentCore is designed for painless installation. Drop the JAR in your plugins folder, restart your server, and you're ready to create your first apartment.

📖 Full Installation Guide
1
Install Dependencies Vault, WorldGuard, and a Spigot/Paper server ≥ 1.21.4
2
Drop the JAR Place ApartmentCore.jar in your /plugins folder
3
Restart & Configure Edit config.yml, create a WorldGuard region, and run /ac admin create
4
Go Live! Players can now purchase, upgrade, and trade apartments
