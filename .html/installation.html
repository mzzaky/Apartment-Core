Dependencies

Required Setup

Before installing ApartmentCore, ensure your server meets all the requirements below. The plugin requires specific dependencies to function properly.

Required Dependencies

Dependency Version Description
Minecraft Server 1.21.4 Latest Minecraft version supported
Spigot/Paper Latest for 1.21.4 Paper is recommended for better performance
Vault 1.7+ Economy API for transaction handling
WorldGuard 7.0+ Region management for apartment areas

Optional Dependencies

Dependency Version Description
PlaceholderAPI 2.11+ Enables placeholder support for signs, scoreboards, etc.

Installation Steps

Step by Step

Follow these steps carefully to ensure a smooth installation process. Make sure you have completed all dependency requirements before proceeding.

1
Download ApartmentCore
Download the latest ApartmentCore.jar file from the official repository or SpigotMC page. Ensure you're downloading the version compatible with your server.
2
Install to Plugins Folder
Place the downloaded ApartmentCore.jar file in your server's plugins folder. Do not rename the file.
3
Install Required Dependencies
Ensure Vault and WorldGuard are installed in your plugins folder. Download them from their official sources if you haven't already.
4
Restart Your Server
Perform a complete server restart (not reload) to properly initialize the plugin and generate configuration files.
5
Configure the Plugin
Navigate to plugins/ApartmentCore/config.yml and adjust settings according to your server's needs. Default settings work for most servers.

Creating Your First Apartment

Admin Guide

Once ApartmentCore is installed and configured, you can start creating apartments. This process requires admin permissions and involves using WorldGuard regions.

Prerequisites

  • You must have operator permissions or the apartmentcore.admin permission
  • WorldGuard must be properly installed and configured
  • The economy system (through Vault) must be functioning

Step-by-Step Creation Process

1
Create a WorldGuard Region
Use WorldGuard's wand tool to select the apartment area, then create a region: 
//wand
//pos1
//pos2
/region define apartment_region
2
Use the Create Command
Execute the apartment creation command with the following syntax: 
/apartmentcore admin create <region_name> <apartment_id> <price>
3
Example Creation
Here's a practical example of creating your first apartment: 
/apartmentcore admin create apartment_region my_first_apartment 99999
This creates an apartment with ID "my_first_apartment" in the "apartment_region" WorldGuard region, priced at 99,999 currency units.
üí° Naming Convention
Use descriptive IDs for your apartments (e.g., downtown_suite_01, beachfront_villa_a). This makes management easier as your apartment system grows.

Verification

Testing

After installation and creating your first apartment, verify everything is working correctly:

Check Plugin Status

/apartmentcore info

List All Apartments

/apartmentcore admin list

Test Purchase System

Have a test player attempt to purchase the apartment to ensure the economy integration is working:

/apartmentcore buy my_first_apartment

Troubleshooting

Common Issues

Plugin Not Loading

  • Check Java Version: Ensure you're using Java 21 or higher
  • Verify Dependencies: Check that Vault and WorldGuard are loaded before ApartmentCore
  • Review Console: Look for error messages during server startup

Cannot Create Apartments

  • Permissions: Verify you have apartmentcore.admin permission
  • Region Exists: Ensure the WorldGuard region is properly defined
  • Unique IDs: Check that the apartment ID isn't already in use

Economy Not Working

  • Vault Installation: Confirm Vault is properly installed
  • Economy Plugin: Ensure you have an economy plugin that supports Vault
  • Player Balance: Verify players have sufficient funds for purchases
