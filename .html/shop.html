Overview

Premium Features

The ApartmentCore Shop System introduces a comprehensive upgrade marketplace where players can enhance their apartments with premium amenities. Each upgrade provides meaningful buffs that directly impact gameplay, from increased income generation to reduced tax burdens.

This system adds depth to property management by allowing players to invest in their apartments for long-term benefits. All upgrades are apartment-specific, meaning each property can be customized independently to suit different strategies and playstyles.

Key Features
5 Unique Categories: Each offering distinct strategic advantages
5 Progressive Tiers: Upgrade from basic to premium levels
Permanent Buffs: Benefits last until apartment ownership changes
Smart Investment: 50% refund when selling your apartment
How It Works
Shop upgrades are tied to individual apartments. When you purchase an upgrade, it only affects that specific property. If you sell the apartment, the new owner starts fresh, but you receive 50% of your total shop investment as a refund. This creates a balanced economy where upgrades add value without making apartments prohibitively expensive.

Visual Indicators

  • Material Types: Shop items display using materials that match their tier (Iron, Gold, Emerald, Diamond, Netherite)
  • Enchantment Glow: Active upgrades feature a glowing effect to show they're currently benefiting your apartment

Strategic Tier Progression
The tier system encourages strategic investment planning. Early tiers are affordable and provide good value, while higher tiers require significant investment but offer powerful bonuses. Consider your apartment's income and tax situation when deciding how far to upgrade each category.

The shop system enforces specific rules to ensure fair gameplay and prevent exploits. Understanding these rules will help you make the most of your shop investments.

üìä Sequential Tiers You must purchase tiers in order (I ‚Üí II ‚Üí III ‚Üí IV ‚Üí V). No tier skipping allowed.

Buff Stacking

Multiple shop items can be purchased for the same apartment, and their effects stack appropriately:

  • Percentage Buffs: Multiple percentage increases are multiplicative (Premium Kitchen tiers stack)
  • Flat Buffs: Flat bonuses are additive (Luxury Furniture provides cumulative increases)
  • Mixed Categories: Different shop categories all apply simultaneously
Investment Planning
Shop upgrades represent significant investments. Before purchasing, ensure your apartment generates enough income to justify the cost. Higher-tier upgrades can take time to pay for themselves through increased earnings.

Refunds & Selling

Economics

The shop system includes a fair refund mechanism that protects your investment while maintaining economic balance. When you sell your apartment, you receive a partial return on your shop investments.

50% Refund Rule

When an apartment changes ownership, the previous owner receives exactly 50% of their total shop investment back. This applies regardless of how the ownership transfer occurs.

Example Calculation
If you've invested $25,000 total in shop upgrades:
‚Ä¢ Premium Kitchen Tier III: $15,000
‚Ä¢ Solar Panel Tier II: $10,000

Automatic Reset System

When apartment ownership changes, several automatic processes occur:

  • Previous Owner: Receives 50% refund of total shop investment
  • Shop Data Reset: All upgrades are removed from the apartment
  • Clean Slate: New owner starts with zero shop upgrades
  • Stats Reset: Ratings, guestbook, and other personalized data are cleared

Investment Protection
The 50% refund system ensures that shop upgrades never completely lose their value. Even if you need to sell an apartment, you recover half of what you spent. This encourages players to invest in upgrades without fear of total loss.

Permissions

Administration

The shop system uses a comprehensive permission structure that allows fine-grained control over who can access and use shop features.

Player Permissions

apartmentcore.shop:
  description: Allow using the apartment shop system
  default: true
  children:
    apartmentcore.shop.buy: true
    apartmentcore.shop.view: true

Administrative Permissions

apartmentcore.admin.shop:
  description: Allow managing shop configurations
  default: op
Permission Node Default Purpose
apartmentcore.shop true Base shop access (includes view and buy)
apartmentcore.shop.view true View shop items and prices
apartmentcore.shop.buy true Purchase shop upgrades
apartmentcore.admin.shop op Manage shop configurations and settings
Permission Management
By default, all players can view and purchase from the shop. If you want to restrict shop access (for example, to premium members only), remove the default permissions and grant them selectively through your permissions plugin.

Configuration File

Shop settings are managed through shop.yml which includes:

  • Customizable costs for each tier
  • Buff values and percentages
  • GUI appearance settings
  • Message customization
  • Performance options
