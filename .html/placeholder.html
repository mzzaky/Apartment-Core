Player Player-Specific Placeholders
These placeholders provide information about specific players and their apartment ownership.
Placeholder Description Output Type
%apartmentcore_owned_count%
 Returns the number of apartments owned by the player Number
%apartmentcore_total_income%
 Returns total pending income from all apartments owned by the player Currency Format
%apartmentcore__shop_total_investment%
 Total money spent on shop items for this apartment Currency Format
%apartmentcore__shop_income_bonus%
 Total income bonus percentage from shop upgrades Percentage
%apartmentcore__shop_base_income%
 Total flat income bonus from shop upgrades Currency Format
%apartmentcore__shop_tax_reduction%
 Total tax reduction percentage from shop upgrades Percentage
%apartmentcore__shop_income_speed%
 Income speed bonus (tick reduction) from shop upgrades Number
%apartmentcore__shop_max_messages%
 Extra guestbook messages from shop upgrades Number
%apartmentcore__shop__tier%
 Tier level of specific shop item (premium_kitchen, luxury_furniture, etc.) Number / 0
%apartmentcore__shop__value%
 Buff value of specific shop item Number / Currency / Percentage
Apartment Apartment-Specific Placeholders
Placeholders provide information about specific apartments attribute. Remember to replace <apartment_id> with a real apartment ID.
Placeholder Description Output Type
%apartmentcore_<apartment_id>_owner%
 Name of the apartment Owner Player Name / "For Sale"
%apartmentcore_<apartment_id>_displayname%
 Display name of the apartment Text String
%apartmentcore_<apartment_id>_price%
 Price of the apartment Currency Format
%apartmentcore_<apartment_id>_tax%
 Legacy base tax amount for the apartment (invoice-based billing is reflected via tax_status and next_invoice_in) Currency Format
%apartmentcore_<apartment_id>_level%
 Level/tier of the apartment Number
%apartmentcore_<apartment_id>_income%
 Pending income for the apartment Currency Format
%apartmentcore_<apartment_id>_status%
 Status of the apartment Active / Inactive
%apartmentcore_<apartment_id>_rating%
 Average rating of the apartment (1 decimal) Number / "N/A"
%apartmentcore_<apartment_id>_welcome%
 Welcome message for the apartment Text String
%apartmentcore_<apartment_id>_tax_status%
 Tax status of the apartment ACTIVE | OVERDUE | INACTIVE | REPOSSESSION | For Sale
%apartmentcore_<apartment_id>_income_in%
 Time until next income generation; returns N/A when not owned or inactive Time Format / Now / N/A
%apartmentcore_<apartment_id>_next_invoice_in%
 Time until next invoice; returns N/A when not owned Time Format / N/A
Statistic Statistic Placeholders
Format: %apartmentcore_statistic_<apartment_id>_<stat_name>%
Stat Name Description Output Type
total_tax_paid Total tax paid for the apartment Currency Format
total_income_generated Total income generated for the apartment Currency Format
ownership_age_days Number of days since apartment ownership began Number (Days)

Note: If a placeholder is invalid or data is unavailable, it may return fallbacks such as "N/A", "Invalid Placeholder", "Unknown Statistic", "For Sale", or "0".

Make sure to replace <apartment_id> with the actual apartment ID when using placeholders.

All currency values are formatted according to the server's economy plugin configuration.
