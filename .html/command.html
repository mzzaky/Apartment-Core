General Commands

General
Essential commands available to all players for basic apartment interactions and information viewing.
Command Description Permission

 Opens the main GUI menu for easy feature access apartmentcore.gui
/apartmentcore buy <id>
 Buy an apartment apartmentcore.buy
/apartmentcore sell <id>
 Sell your apartment apartmentcore.sell

 Confirm pending action apartmentcore.use

 List apartments apartmentcore.list
/apartmentcore version
 Check plugin version apartmentcore.use
/apartmentcore rate <id> <0-10>
 Rate an apartment apartmentcore.use

Guestbook Commands

Guestbook
Commands for interacting with apartment guestbooks. Leave messages, read feedback, and manage your guestbook entries.
Command Description Permission
/apartmentcore guestbook leave <id> <message>
 Leave a message in a guest book apartmentcore.use
/apartmentcore guestbook read <id>
 Read your guest book messages apartmentcore.use
/apartmentcore guestbook clear <id>
 Clear all messages from your guest book Owner only

Owner Commands

Owner
Exclusive commands for apartment owners to manage their properties, including income collection, tax payment, and customization options.

Command Syntax

  • <id> - Required apartment ID parameter
  • <name> - Custom name for your apartment
  • <message> - Custom welcome message text
Command Description Permission
/apartmentcore teleport <id>
 Teleport to your apartment apartmentcore.teleport
/apartmentcore upgrade <id>
 Upgrade apartment level apartmentcore.upgrade
/apartmentcore rent claim <id>
 Claim generated income apartmentcore.rent
/apartmentcore rent info <id>
 View income information apartmentcore.rent
/apartmentcore tax info
 View tax information for all your apartments apartmentcore.tax
/apartmentcore tax pay
 Pay all outstanding taxes for all your apartments apartmentcore.tax
/apartmentcore tax auto <on|off>
 Enable or disable automatic tax payments apartmentcore.tax
/apartmentcore setname <id> <name>
 Set apartment display name Owner only
/apartmentcore setwelcome <id> <message>
 Set welcome message Owner only
/apartmentcore setteleport <id>
 Set teleport location apartmentcore.setteleport

Auction Commands

Auction
Commands for interacting with the apartment auction system. Create auctions, place bids, and view the market.
Command Description Permission
/apartmentcore auction create <id> <bid> <hours>
 Create an auction for your apartment apartmentcore.auction.create
/apartmentcore auction bid <id> <amount>
 Place a bid on an apartment auction apartmentcore.auction.bid
/apartmentcore auction list [filter]
 List all active auctions (filters: mine, ending, nobids) apartmentcore.auction.list
/apartmentcore auction cancel <id>
 Cancel your own auction (if no bids) apartmentcore.auction.cancel

Admin Commands

Admin
Powerful administrative commands for server staff to manage all apartments, create new properties, modify settings, and maintain the plugin.

Admin Command Parameters

  • <region> - WorldGuard region name for the apartment
  • <id> - Unique ID for the apartment
  • <player> - Target player name
  • <time> - Time period in seconds
  • <level> - Apartment level (1-10)
  • <filename> - Backup file name for restoration
Command Description Permission
/apartmentcore admin create <region> <id> <price>
 Create an apartment from a WorldGuard region apartmentcore.admin
/apartmentcore admin remove <id>
 Remove an apartment apartmentcore.admin
/apartmentcore admin set <property> <id> <value>
 Set a property of an apartment (owner, price, level, rate) apartmentcore.admin
/apartmentcore admin status <id> <active|inactive>
 Manually set an apartment's status apartmentcore.admin
/apartmentcore admin invoice <add|remove> <id> <amount|invoice_id>
 Manage tax invoices for an apartment apartmentcore.admin
/apartmentcore admin apartment_list
 List all apartments apartmentcore.admin
/apartmentcore admin teleport <id>
 Teleport to any apartment apartmentcore.admin
/apartmentcore admin backup <create|list|restore> [filename]
 Manage plugin data backups apartmentcore.admin
/apartmentcore admin reload
 Reload the plugin's configuration apartmentcore.admin
/apartmentcore admin auction list [filter]
 List all auctions with detailed admin info apartmentcore.admin
/apartmentcore admin auction cancel <id>
 Force-cancel any auction and refund bidders apartmentcore.admin
/apartmentcore admin auction forceend <id>
 Force-end an auction and process the winner apartmentcore.admin

Note: Command parameters enclosed in <angle brackets> are required, while those in [square brackets] are optional.

Do not include the brackets when executing the commands.
